# NCBT - Nursing Computer Based Test Exams

Welcome to **NCBT** - a new age destination for Nursing competitive exam Computer Based Tests. This platform provides comprehensive mock tests designed to help nursing professionals prepare for their competitive exams.

🌐 **Visit us at:** [https://ncbt.in](https://ncbt.in)

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## About NCBT

NCBT provides a comprehensive platform for nursing professionals to practice and excel in their competitive exams through interactive Computer Based Tests.
